1. Regular expressions

Regular expressions are one of the ways to process character strings and are a means to make the process of’searching’ or’substituting’ characters in a specific condition very simple.

2. Term

  • ^x : x 문자로 시작

  • x$ : x 문자로 종료

  • .x : 임의의 한 문자의 자리수를 표현하며 문자열이 x 로 끝남

  • x+ : x 문자가 1개 이상

  • x? : x 문자가 0개 ~1개

  • x* : x 문자가 0개 ~ 여러개

  • x|y : or 를 표현하며 x 또는 y 문자

  • (x) :그룹을 표현하며 x 를 그룹으로 처리

  • (x)(y):x 그룹을 찾고 그 뒤에 있는 y값을 찾음

  • x.?? : 앞에 있는 ?의 최소값인 0개로 검색 (? lazy)

  • x.+? : 앞에 있는 +의 최소값인 1개로 검색 (? lazy)

  • x+(?=y) : 끝이 y로 끝나는 x를 검색

3. Example

  • 전화번호 : ([\d]{3}) \d{3}-\d{4} 

    \([0-9]{3}\) [0-9]{3}-[0-9]{4}

  • 전화번호 (중간에 8또는 1 제외): [^8|^1][0-9]{2}-[0-9]{4}
