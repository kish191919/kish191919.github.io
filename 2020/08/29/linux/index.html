1.Command

  • ls : 현재 디렉토리의 파일 목록을 출력하는 명령어.

    ls -l' : 자세히 보기

    ls -al : 숨겨진 파일 보기 & 자세히 보기

    ls -S : 큰사이즈 파일 순 으로 정렬

  • pwd

  • mkdir
    mkdir -p dir1\dir2\dir3 : 여러디렉토리를 한번에 만들수 있고, 없는 디렉토리라면 추가해

  • cd

    cd .. : 현재 디렉토리의 부모 디렉토리로 이동

  • rm

    rm -r : 폴더 삭제 / r은 재귀적이란 뜻으로 폴더안에 폴더 등을 순차적으로 지운다라는 의미

  • –help : 명령어 뒤에 –help를 붙이면 명령의 사용설명서 출력 (간단히 출력)

    ls --help

  • man : 명령어 앞에 man을 붙이면 명령문 사용설명서 출력 (화면 빠져나와서 자세히 출력)

    man ls : ls 명령어를 자세히 살펴볼수 있고, /sort 는 sort를 검색

    n 키 : 페이지 넘기
    q 키 : 밖으로 빠져나가기

  • clear

  • touch : 파일 생성

  • cp : 파일 복사

  • mv : 파일 이동 및 이름변경

  • sudo : - super user do 의 줄임말

    - 예전에는 하나의 컴퓨터로 여러명이 사용을 했고 누군가는 강력한 권한을 갖고 명령어를 사용

  • top : 사용중인 프로그램 목록 및 용량

  • head -n1 text_file: 1줄만 출력

  • head -n1 text_file > one.txt : 출력값을 one.txt에 다시 저장

2. 유용한 command

  • 명령어 횟수 : history | awk ‘{a[$2]++}END{for(i in a){print a[i] “ “ i}}’ | sort -rn | head -10

3. APT - Package Manager (Mac의 경우 brew 사용)

  • sudo apt-get update : 최신 상태의 소프트웨어 목록 다운로드

  • sudo apt-cache search 프로그램 : 현재 목록 상태에서 원하는 프로그램을 찾음

  • sudo apt-get install 프로그램 : 현재 목록에있는 프로그램 설치

  • sudo apt-get upgrade 프로그램 : 프로그램을 업그레이드 함

  • sudo apt-get upgrade : 설치된 모든 프로그램을 업그레이드 함

  • sudo apt-get remove 프로그램 : 프로그램 삭제

4. wget - 다운로드 (Mac의 경우 curl 사용)

  • wget 다운로드주소 : 다운로드 주소에 있는 파일 저장 [curl 다운로드주소]

  • wget -O cow.jpg 다운로드주소 : 다운받은 파일을 cow.jpg 파일로 저장 [curl -o cow.jpg 다운로드주소]

5. git

  • git clone 다운받는주소 새디렉토리이름: 새디렉토리가 만들어지고 그 안에 git이 다운로드 됨

6 pipeline

  • ps aux | grep sort : 프로세스 중 sort로 된 것을 찾음

7. IO redirection

  • ls -al > result.log : ls -al의 결과를 result.log에 저장 (ls -al 1> result.log )

  • rm result.log 2> error.log : error message 가 error.log에 저장. > 전에 반드시 2를 붙임

1 : standard output / 2 : standard error

rm result.log 1> result.log 2> error.log

8. Shell Script

/bin : 기본 프로그램의 명령어가 위치 (명령어가 파일 형식으로 되어있)

nano backup : backup 파일 작성

#!/bin/bash : bash 안에 있는 명령문 사용가능

if [ -d bak ] : 현재 디렉토리에 bak 디렉토리가 존재한다면

if ! [ -d bak ]: 현재 디렉토리에 bak 디렉토리가 존재하지 않는다면

chmod +x backup : 파일에 실행 권한 주기

./backup : backup 스크립트 실행
