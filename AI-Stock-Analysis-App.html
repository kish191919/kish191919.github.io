AI Stock Analysis App

📈 AIStockAnalysisApp

Subject : AI-powered real-time stock analysis iOS application

Tech : Swift, SwiftUI, Core Data, URLSession, JSONDecoder, Yahoo Finance API, OpenAI GPT-4

Features : Real-time stock data collection, AI-driven multi-language investment insights, Analysis history storage, Cost tracking

📌 Overview

AIStockAnalysisApp is an iOS application that leverages SwiftUI and OpenAI GPT-4 to analyze real-time stock market data and deliver actionable investment insights in multiple languages.
It fetches market data from the Yahoo Finance API, processes analysis via OpenAI, and stores analysis history locally using Core Data.

🚀 Key Features

  1. Real-time Stock Data Collection

    • Fetches 3 days of intraday (15-min interval) and 1-month daily data from Yahoo Finance API
    • Includes pre-market, regular, and after-hours prices
    • Retrieves latest news and market sentiment indicators (VIX, Fear & Greed Index)

  2. AI-powered Multi-language Stock Analysis

    • Provides:
      • Investment decision: BULLISH, BEARISH, or NEUTRAL
      • Confidence percentage
      • Detailed reasoning in the selected language
      • Predicted next-day closing price
    • Tracks API token usage and calculates cost in real time

  3. Analysis History Storage

    • Saves analysis details locally via Core Data, including:
      • Symbol, current price, decision, confidence, expected price, reason, language, timestamp

  4. User-friendly UI

    • TabView with:
      • Home – Search and view live stock data
      • Analysis – Run AI-powered analysis
      • History – Review past analysis

📊 OpenAI API Cost Calculation

  • GPT-4 Turbo Pricing (as of 2024):
    • Prompt tokens: $0.01 / 1K tokens
    • Completion tokens: $0.03 / 1K tokens
  • Each API call logs token usage and calculates cumulative cost

🖥 How to Run

  1. Set API Key

  2. Build & Run in Xcode

    • Select iPhone or simulator
    • Click Run to launch the app

  3. Perform Analysis

    • Enter a stock symbol → Fetch data → View AI analysis results

📌 Future Improvements

  • Add more technical indicators (e.g., MACD, RSI)
  • Enhance multi-language UI support
  • Add user-customized alerts
  • Optimize token usage to reduce API costs
