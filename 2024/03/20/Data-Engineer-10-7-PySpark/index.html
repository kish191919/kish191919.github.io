2. Zeppelin 환경구성

(1) zeppelin 다운로드 - spark-master-01

  • Binary package with all interpreters

(2) zeppelin 설정 - spark-master-01

  • ip 변경 127.0.0.1 => 0.0.0.0
  • port 변경 8080 => 9090

(3) zeppelin 실행 (on spark-master-01)

(4) zeppelin web ui 접속 (필요시 방화벽 설정)

  • http://spark-master-01:9090
  • 필요시 방화벽 오픈 - 9090 0.0.0.0/0 Zeppelin Web UI

(5) spark interpreter 설정 (기존 변경)#01 (spark -> master: local[*])

  • interpreter 관리 화면

    • zeppelin web ui > anonymous > Interpreter 메뉴 클릭

  • interpreter 조회

    • spark

  • interpreter 정보 수정

    • edit 버튼 클릭

  • interpreter 정보 입력

(6) spark interpreter 설정 (신규 생성) - Spark Standalone

  • spark_stand -> master: spark://spark-master-01:7177

  • interpreter 관리 화면

    • zeppelin web ui > anonymous > Interpreter 메뉴 클릭

  • interpreter 생성

    • +Create 버튼 클릭

  • interpreter 정보 입력

(7) Spark Standalone 클러스터 매니저 실행

(8) Standalone 웹UI 확인

  • http://spark-master-01:8180

(9) spark interpreter 설정 (신규 생성) - YARN

  • spark_yarn -> master: yarn

  • interpreter 관리 화면

    • zeppelin web ui > anonymous > Interpreter 메뉴 클릭

  • interpreter 생성

    • +Create 버튼 클릭

  • interpreter 정보 입력

(10) YARN 클러스터 매니저 실행 (+ HDFS 실행)

(11) YARN 웹UI 확인

  • http://spark-master-01:8188

(12) HDFS 웹UI 확인

  • http://spark-master-01:9870

(13) new note 생성 (data_Interpreter_Test)

  • 신규 노트 생성

    • zeppelin web ui > Notebook > +Create new note 메뉴 클릭

  • 노트 설정

    • Note Name => data_Interpreter_Test
    • Default Interpreter => spark
    • 맨 위에 있는 Interpreter가 디폴트가 됨

  • Interpreter 바인딩 확인

    • 기어 아이콘 클릭

(14) interpreter test#01 (spark)

  • paragraph 코드 작성 및 실행 (삼각형 버튼 클릭 or “Shift + Enter”)
  • 프로세스 확인(Java 프로세스)
  • 프로세스 확인(Python 프로세스)

(15) interpreter test#02 (spark_stand)

  • paragraph 코드 작성 및 실행 (삼각형 버튼 클릭 or “Shift + Enter”)
  • 프로세스 확인(Java 프로세스)
  • 프로세스 확인(Python 프로세스)
  • Standalone 웹UI 확인
    • http://spark-master-01:8180

(16) interpreter test#03 (spark_yarn)

  • paragraph 코드 작성 및 실행 (삼각형 버튼 클릭 or “Shift + Enter”)
  • 프로세스 확인(Java 프로세스)
  • 프로세스 확인(Python 프로세스)

  • YARN 웹UI 확인

    • http://spark-master-01:8188

  • HDFS 웹UI 확인

    • http://spark-master-01:9870
