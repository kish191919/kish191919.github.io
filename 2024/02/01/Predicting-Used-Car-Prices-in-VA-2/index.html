2. Pickle로 패스워드 저장하기

  • ‘pickle’은 파이썬 객체를 파일로 저장하고 불러오는데 쓰이는 모듈

    • 예를 들어 작업 중인 파이썬 객체를 저장해 두고 나중에 이어서 작업하고 싶을 경우 사용하면 좋은 모듈

      • 파일 저장하기: 피클링할 경우 (파이썬 객체 → 파일) pickle.dump(object, file)
      • 파일 불러오기: 언피클링할 경우 (파일 → 파이썬 객체) pickle.load(file)
