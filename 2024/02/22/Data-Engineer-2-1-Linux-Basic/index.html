linux_basic

  • man

    • 원하는 명령어의 설명 (Manual) 을 볼수 있음
      • ex) man find : find 명령어 볼수 있음
    • 기본 명령어가 아닌경우 -help, –help의 파라미터로 설명을 제공하는 명령어가 있음 -> 화면에 출력되기때문에 지저분할수 있음
    • J - 다운, K - 업 , Q - 종료, H - help

  • cd

    • cd .. : 상위 디렉토리로 이동
    • cd etc/ 다음에 탭을 2번 연달아 눌러주면 모든 서브 디렉토리가 표시
    • cd - : 이전에 갔던 디렉토리로 되돌아감

  • pwd

    • 절대 경로를 표시

  • ls

    • ls -il : 파일 또는 디렉토리의 inode number를 표시
      • inode란 리눅스에서 파일 또는 디렉토리의 고유 식별 번호로서 파일 이름이 바뀌어도 유지
    • 파일 숨기기 (파일이름 앞에 .을 붙이면 됨) : mv hello.txt .hello.txt

  • df

    • df -h : 디스크 공간 정보 , 인간이 읽을 수 있음 (h)

  • du

    • du -sh : 해당경로 하위의 총합 (s), 인간이 읽을 수 있음 (h)
    • du -h -d 2 : 해당 depth 만큼 보여줌

  • chmod

    • chmod +x test.txt : test 파일의 오너, 그룹, other 모두 x 가 추가
      • 실행 : ./test (앞에 ./ 를 붙여서 실행)
    • chmod u+x test.txt : test 파일의 오너만 x 가 추가
    • chmod g+w,o-r test.txt : test 파일의 그룹 w 가 추가, other에게는 r를 제거.
    • chmod 760 test.txt

  • head

    • head -n 3 : 첫 3줄만 출력

  • tail

    • tail -f

  • comm

    • comm -12 line1.txt line2.txt : line1 과 line2 공통 라인 출력
    • comm -3 line1.txt line2.txt : line1 과 line2 다른 라인 출력

  • cmp

    • 두 파일을 바이트 단위로 비교한 결과를 출력

  • diff

    • 두 파일의 라인별 차이점을 보여줌
    • symbol 차이
      • a : add
      • c : change
      • d : delete

  • less

    • 터미널 세션에 프린트하지 않고 파일내용 볼수있음
    • 보는 부분만 열기때문에, 큰 용량의 파일을 볼때는 vi보다 빠름
    • 양방향 탐색이 가능하기때문에 more보다 편리
      • j : down 5j 다섯줄 아래 space : 한 화면씩 아래
      • k : up 5k 다섯줄 위 b : 한 화면씩 위
      • shift + g : 맨아래
      • h : help

  • ln

    • 심볼릭 링크 : 절대경로 또는 상대경로 형태로 된 다른 파일이나 디렉토리에 대한 참조를 포함한 특별한 종류의 파일
    • in -s $원래파일위치 $목표파일위치
    • 원래 파일을 삭제하면 심볼릭 링크로 만든 파일에서 에러가 발생
    • 파일에 심볼릭 링크 설정
    • 폴더에 심볼릭 링크 설정

  • alias

    • 다른 문자열에 대채하는 단어 지정
    • 주로 복잡한 명령어의 약어로 사용
    • 환경변수와 다름

  • cp

    • cp -r 디렉토리 타겟디렉토리 :

  • find

    • find . -name 파일이름 : 파일을 찾음
    • find . -name ‘정규표현식’

  • which

    $PATH에 등록된 경로중에 주어진 이름의 실행 파일의 위치 보여줌
    사용하는 명령어의 위치 찾음

  • grep

    grep -rl “[0-9]0{3}“ . : 현재 디렉토리에서 하위 디렉토리까지 검색하여 맞는 조건의 파일이름만 표기
    grep -n “[0-9]0{1}“ find.txt : 찾는 문자열의 줄번호 표기

  • sed

    텍스트를 필터링하거나 변환하는 스트림 에디터
    sed s/old_word/new_word/g target_file : 타겟파일에서 old word를 new word 로 교체

    vi에서도 %s/old_word/new_word/g 를 이용하여 단어를 교체
    :%s/where/find/g

uname

  • 이름, 버전, 기타 시스템 정보확인
  • -a : 모든정보 표기
