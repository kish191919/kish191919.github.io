2024-02-29
2024-02-29
linux_basic, vi
  1. vi

    • set number

    • set nonumber

    • 방향키

      h : 왼쪽
      l : 오른쪽
      j : 아래
      k : 위
      w : 다음 단어 첫글자 이동
      b : 이전 단어 첫글자 이동

    • 행이동

      gg : 맨 앞장 첫행으로 이동
      shift + g : 맨 마지막으로 이동
      ^ : 현재행의 첫 문자로 이동
      0 : 현재행의 맨 앞으로 이동
      $ : 현재행의 마지막 문자로 이동
      :5 : 타이핑한 숫자행으로 이동
      숫자 타이핑하고 shift + g : 타이핑한 숫자행으로 이동 (위와 동일)
      숫자 타이핑하고 엔터 : 해당숫자만큼 아래로 이동

    • 문자 검색을 이용한 이동 (정규식 이용 가능)

      /문자열 : 현재 위치에서 문자열을 검색하고 이동
      ?문자열 : 현재 위치에서 이전부분으로 문자열을 검색하고 이동
      n : 위에 명령어를 반복해서 실행
      N : 역방향으로 반복해서 실행

    • 입력 관련 명령어

      i : 현재 커서 위치에서 글자 삽입 가능
      a : 현재 커서 다음 위치에 추가 (append 의미)
      I : 현재 줄 처음 글자에 삽입
      A : 현재 줄 마지막 글자에 추가
      o : 아랫 줄에 추가
      O : 윗줄에 추가
      s : 현재 커서 글자 지우고 입력모드로 전환
      r : 현재 커서 글자 지우고 한 글자 입력 받아 바꾼뒤 이동 모드로 변경
