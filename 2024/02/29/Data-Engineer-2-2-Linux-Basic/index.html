  1. vi

    • set number

    • set nonumber

    • 방향키

      h : 왼쪽
      l : 오른쪽
      j : 아래
      k : 위
      w : 다음 단어 첫글자 이동
      b : 이전 단어 첫글자 이동

    • 행이동

      gg : 맨 앞장 첫행으로 이동
      shift + g : 맨 마지막으로 이동
      ^ : 현재행의 첫 문자로 이동
      0 : 현재행의 맨 앞으로 이동
      $ : 현재행의 마지막 문자로 이동
      :5 : 타이핑한 숫자행으로 이동
      숫자 타이핑하고 shift + g : 타이핑한 숫자행으로 이동 (위와 동일)
      숫자 타이핑하고 엔터 : 해당숫자만큼 아래로 이동

    • 문자 검색을 이용한 이동 (정규식 이용 가능)

      /문자열 : 현재 위치에서 문자열을 검색하고 이동
      ?문자열 : 현재 위치에서 이전부분으로 문자열을 검색하고 이동
      n : 위에 명령어를 반복해서 실행
      N : 역방향으로 반복해서 실행

    • 입력

      i : 현재 커서 위치에서 글자 삽입 가능
      a : 현재 커서 다음 위치에 추가 (append 의미)
      I : 현재 줄 처음 글자에 삽입
      A : 현재 줄 마지막 글자에 추가
      o : 아랫 줄에 추가
      O : 윗줄에 추가
      s : 현재 커서 글자 지우고 입력모드로 전환
      r : 현재 커서 글자 지우고 한 글자 입력 받아 바꾼뒤 이동 모드로 변경

    • 삭제

      x : 현재 커서 위치 문자 삭제
      X : 현재 커서 위치 이전 문자 삭제
      dw : 현재 커서 위치 단어 삭제
      db : 현재 커서 이전 단어 삭제
      dd : 현재 커서 위치 줄 삭제
      d^ : 현재 줄에서 현재 커서 위치 이전 문자열 모두 삭제
      d0 : 위의 명령어와 동일 그러나 컬럼 0번째 자리까지 삭제
      d$ : 현재 커서 위치에서 줄의 마지막 까지 삭제

    • 복사

      y : 복사
      yw : 현재 커서 이후 단어 복사
      yb : 현재 이전 단어 복사
      yy : 현재 줄 복사
      p : 붙여넣기
      P : 커서 위치 이전에 붙여넣기

    • 되돌리기

      u : undo
      Ctrl + r : redo

    1. shell script

      • 파일 맨 위에 다음과 같은 표시가 있다면 shell script 파일
      • 실행방법 : ./$Filename 을 이용, 만약 안된다면 chmod +x 로 권한 주기
      • 변수 : 영문, 숫자, _ 만 사용
      • 변수명은 대문자로 작성

      • 특수 목적 변수

        • $ : 현재 shell의 프로세스 아이디
        • $0 : 현재 스크립트의 파일 이름
        • $n : 스크립트 실행시 넘겨준 n번째 인자
        • $# : 스크립트에 넣어준 인자의 개수
        • $* : 모든 인자를 “”로 감싸서 반환
        • $@ : 각 인자를 “”로 감싸서 반환
        • $? : 마지막으로 실행된 명령어의 종료 상태

      • $! : 마지막 백그라운드 명령어의 프로세스 아이디
